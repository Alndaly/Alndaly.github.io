## src目录下框架目录结构（webpack底层框架版本）

| 目录            | 内容                                   |
| --------------- | -------------------------------------- |
| pages           | 存放网站页面级别组件                   |
| components      | 存放网站普通组件级别组件               |
| utils           | 存放网站操作函数                       |
| constant        | 存放网站操作用到的常量                 |
| router          | 存放网站路由设计                       |
| layout          | 存放网站整体布局设计组件               |
| static          | 存放网站静态资源                       |
| public          | 存放网站公共目录                       |
